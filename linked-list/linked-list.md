### 연결 리스트 (Linked List)

<pre style="font-family: monospace;">
┌─────┐    ┌─────┐    ┌─────┐
│ val │ -> │ val │ -> │ val │ -> None
│  1  │    │  2  │    │  3  │
│next │ -> │next │ -> │next │
└─────┘    └─────┘    └─────┘
</pre>

- val : 노드가 저장하는 값
- next : 다음 노드를 가리키는 포인터
- 마지막 노드의 next는 None으로 표시
